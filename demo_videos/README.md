# Demo Videos

Place demo videos here for easy access and testing.

## Presentation Slides Video

To use the presentation slides video:

1. Copy your `presentation slides.mp4` file to this directory
2. Rename it to `presentation_slides.mp4` (optional, for convenience)
3. Run the demo:
   ```bash
   python semantic_diff_demo.py demo_videos/presentation_slides.mp4
   ```
